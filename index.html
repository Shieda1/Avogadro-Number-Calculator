What is the avogadro number?
Avogadro number is the number of units in one mole of a substance. In simple words avogadro's number is the molecular weight of a substance in grams. It's unit can be ions, atoms, molecules or electrons.
How to calculate avogadro number?
To calculate avogadro number you should use the following formula:
number of particles = number of moes * avogadro number
where:
avogadro mumber = 6.023 * 10^23
Example: A substance has 8 moles, calculate the number of particles in this substance:
Using avogadro number formula the answer will be:
number of particles = number of moes * avogadro number
number of particles = 8 * 6.023 * 10^23
number of particles = 48.184 * 10^23 particles
Why use avogadro number?
Avogadro number is used to compare the different atoms of specific substances where the same number of atoms are being compared.
